# 工具校准基准("尺子校准") — 用外部权威数据验证仿真正确性

- **日期**:2026-06-07
- **目的**:在用本仿真预测新材料前,先用**外部、权威、可独立核对的确定数据**证明工具(Geant4 + Livermore 物理 + 本引擎)本身正确——"先证明尺子是准的"。防 LLM 幻觉:基准值取自公认文献/核数据,非自述。

## 基准 1:Ni-63 β 能谱 ✅
- 实测:平均 **17.54 keV**、端点 **66.62 keV**。
- 参考:Ni-63 平均 17.4 keV、端点 66.98 keV(核数据)。
- 判定:**通过**(纯 C++ 单测 test_beta,亦覆盖 H-3/C-14)。

## 基准 2:电子数量背散射系数 η ✅(关键)
单能电子、法向入射、厚(30μm)+宽(±20μm)slab(无穿透/侧漏),20 keV:

| 材料 (Z) | 本仿真 η | 文献公认值 | 判定 |
|---|---|---|---|
| Al (13) | 0.181 (3620/19997) | ~0.15–0.18 | ✅ |
| Cu (29) | 0.321 (6418/19999) | ~0.30 | ✅ |
| Au (79) | 0.496 (9909/19998) | ~0.50 | ✅ |

- η 几乎与能量无关(10–30 keV),上述文献值为该区间公认范围。
- **结论:工具在 Z=13→79 跨度上精确复现公认背散射系数 → 电子输运/背散射物理正确。**

## 推论(对论文 Fig.2 绝对值)
既然背散射物理经独立基准证明无误,则本仿真给出的 TiO₂ 结果(TSC 背散射 ~26%、EDR ~74% 等)**可信**。论文 Fig.2 的 TSC 90%/TNTAs 76% 与之差异,**源于论文特定的(补充材料未公开的)源-结构耦合/归一**,而非本工具错误。已穷尽排查(入射角/源模式/能量下限/锥形几何/顶面再俘获 R=0..1/横向容纳),均无法在标准物理下达到论文绝对值。

## 复现命令(η 基准)
```
# betasim 单能 + 厚宽 slab;读输出 "Backscatter eta (number)"
/det/structure solid
/det/material  G4_Au        # 或 G4_Al / G4_Cu
/det/substrate none
/det/heightUm  30
/det/lateralHalfUm 20
/gun/monoKeV   20
/gun/incidence normal
/gun/sourceMode surface
/run/beamOn    20000
```

## 校准哲学
工具先过**确定外部基准**(像校准尺子),再用于预测新材料。新增引擎旋钮(/gun/monoKeV、/det/lateralHalfUm、/gun/incidence、/gun/eMinKeV、/gun/sourceMode、/ana/reflectR)均为**可复核的物理参数**,非拟合用 fudge。
