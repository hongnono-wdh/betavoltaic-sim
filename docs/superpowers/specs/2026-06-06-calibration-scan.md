# 入射角标定扫描记录(批次 2 尽职验证)

- **日期**:2026-06-06
- **目的**:严谨确定"在标准 Geant4 物理 + 论文给定几何下,β 吸收率(EDR 饱和值)的可达天花板",回答"能否复现论文 Fig.2 绝对值"。
- **方法**:三种入射角模型 × {TSC 实心, TNTAs 空心管},Ni-63,各 10000 事件,EDR=沉积/入射(论文口径)。

## 结果

| 入射模型 | TSC EDR 饱和 | TNTAs EDR 饱和 |
|---|---|---|
| isotropic(真实 4π) | 59.3% | 26.8% |
| hemisphere(2π 下行) | 59.3% | 25.9% |
| normal(法向) | **74.4%** | 13.9% |
| **论文 Fig.2** | **90%** | **76%** |

## 结论(铁证)

1. **TSC 天花板 ~74%**(法向入射,低背散射),仍低于论文 90%。
2. **TNTAs 天花板 ~27%**(各向同性,角度入射使 β 多穿管壁),远低于论文 76%。
3. **二者不可兼得**:法向利好实心(74%)却使空心暴跌(14%,β 直穿管腔);各向同性利好空心(27%)却使实心只有 59%。**不存在同时接近 90% 与 76% 的单一物理模型。**
4. 因此论文 Fig.2 的绝对吸收率**无法在标准电子输运 + 论文给定几何下复现**,需论文未公开的源-结构耦合假设(强多次反射陷俘 / 特定角分布 / 或不同的 EDR 归一定义)。

## 已排除的增强尝试(均无法弥合差距)
表面源、Ni 源自吸收、入射能量归一、厚 Ni 阴极反射层、Ti 衬底背散射。Ni/Ti 对低能 β 以吸收为主、反射弱,多次反射陷俘不足以把 TNTAs 推到 76%;TNTAs 的 24% 管壁面积分数从几何上限制了单程吸收。

## 追加:有效源能量下限(束硬化)扫描(法向入射 + 锥形几何)

假设:真实 200nm Ni 源吸收掉软 β → 硬化束 → 降低背散射。新增 `/gun/eMinKeV` 旋钮丢弃软 β,实测:

| eMin (keV) | TSC EDR 饱和 | TNTAs EDR 饱和 |
|---|---|---|
| 0(自然谱) | **74.3%** | **42.6%** |
| 5 | 73.8% | 41.8% |
| 10 | 72.5% | 38.6% |
| 15 | 69.7% | 34.7% |

**反直觉但合理**:硬化束**反而降低 EDR**。因为存在两个竞争损失——软 β 背散射多、**硬 β 穿透多**(近 67keV 端点 β 射程 >12μm,直接穿透 TSC 底部跑掉)。自然谱已接近最优。**束硬化无法逼近论文 90%/76%。**

## 标定总结(穷尽)
已系统排查并实测:入射角(各向同性/下半球/法向)、源位置(体内/表面)、Ni 源自吸收、厚 Ni 阴极反射、锥形 vs 直管几何、有效源能量下限。**天花板:TSC ~74% / TNTAs ~43%**,且 TNTAs 最优高度 10.1μm、电解液 14.7% 已与论文吻合。逐位匹配 90%/76% 需论文**补充材料**(Fig.S1 等,本 PDF 未含)所述的源-结构耦合/归一细节,标准物理 + 正文给定信息下不可达。**不以反推/造假凑数。**
